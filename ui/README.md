# AuthFlow

A modern PHP authentication system with a clean admin dashboard.

## Features

* User Registration
* User Login
* Secure Password Hashing
* User Profile
* Change Password
* Dashboard
* Search Users
* Dark / Light Mode
* Responsive Design
* Toast Notifications

## Technologies

* PHP 8
* MySQL
* JavaScript
* HTML5
* CSS3
* Chart.js
* Font Awesome

## Installation

1. Clone or download the project.
2. Import `authflow.sql` into MySQL.
3. Update `database/database.php`.
4. Start Apache and MySQL.
5. Open the project in your browser.
